# WanderLust Web Application

A travel listing web application built using Node.js, Express, MongoDB, and EJS.

## Features
- Create travel listings
- Upload images
- View listing details
- Edit and delete listings
- User authentication

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Cloudinary
- Bootstrap

## Installation

git clone https://github.com/Netra-sarvade/WanderLust-web-Application.git

cd WanderLust-web-Application

npm install

npm start
